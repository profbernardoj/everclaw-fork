## Revenue Model Categories

### Category 1: SaaS Subscription
**Flavors:** All platform and use case flavors
**Model:** Monthly/annual subscription for premium features

### Category 2: Transaction Fee
**Flavors:** Protocol flavors (BitcoinClaw, EthereumClaw, etc.)
**Model:** Small fee on transactions routed through the agent

### Category 3: Referral Revenue
**Flavors:** All flavors
**Model:** Referral fees from VM setup (StarkClaw) and hardware (IronHill/ClawBox)

### Category 4: MOR Staking
**Flavors:** All flavors
**Model:** User stakes MOR for perpetual inference
