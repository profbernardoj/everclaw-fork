## Demo Use Cases

### Use Case 1: "I'm Leaving"
**Setup:** User is exiting the house
**Command:** "I'm leaving"
**HomeClaw Actions:**
- Turns off all lights
- Locks all doors
- Sets thermostat to away mode
- Arms security system
- Starts robot vacuum
- Sends notification: "Home secured. Vacuum running."

**Why It Works:** One command instead of 6 separate automations

### Use Case 2: "Goodnight"
**Setup:** User going to bed
**Command:** "Goodnight"
**HomeClaw Actions:**
- Turns off all lights except bedroom
- Locks doors
- Sets thermostat to sleep temperature
- Disables motion sensors in bedroom
- Starts white noise
- Schedules gradual wake-up lighting for 6:30 AM

**Why It Works:** Context-aware bedtime routine

### Use Case 3: "Movie Time"
**Setup:** User wants to watch a movie
**Command:** "Movie time"
**HomeClaw Actions:**
- Dims living room lights to 10%
- Closes blinds
- Turns on TV and receiver
- Sets TV to preferred streaming input
- Pauses robot vacuum if running

**Why It Works:** Entertainment mode in one command

### Use Case 4: "Something's Wrong"
**Setup:** User notices something unusual
**Command:** "Why is the living room so cold?"
**HomeClaw Response:**
- Checks thermostat history
- Checks window sensors
- Responds: "The living room window has been open for 2 hours. Temperature has dropped 4 degrees. Want me to close the blinds?"

**Why It Works:** AI diagnoses problems, doesn't just execute commands

### Use Case 5: "Guest Mode"
**Setup:** User has visitors
**Command:** "Guests are here"
**HomeClaw Actions:**
- Creates temporary access codes for smart locks
- Adjusts lighting for guest areas
- Disables sensitive notifications
- Creates guest Wi-Fi QR code
- Sets guest room temperature

**Why It Works:** Automatically handles visitor logistics
