# Flavor Categories — "Which Flavor Are You?" Quiz Concept

**Purpose:** Interactive quiz for InstallOpenClaw.xyz to help users find their flavor

---

## Quiz Structure

### Question 1: What's your primary focus?
- A) Crypto/DeFi → Protocol flavors
- B) AI/ML → AI Model flavors
- C) Productivity/Work → Platform/Use Case flavors
- D) Personal/Family → Use Case flavors

### Question 2: How do you use AI?
- A) For work → OfficeClaw, InvestClaw, VCIClaw, BriefingClaw
- B) For personal life → FamilyClaw, FriendClaw, BookingClaw, EmailClaw
- C) For crypto → BitcoinClaw, EthereumClaw, SolanaClaw, etc.
- D) For development → LinuxClaw, AndroidClaw, WindowsClaw

### Question 3: What matters most?
- A) Privacy → All flavors (local-first)
- B) Cost savings → DeepSeekClaw, MiniMaxClaw
- C) Decentralization → MorpheusClaw, BitcoinClaw
- D) Convenience → AppleClaw, WindowsClaw
